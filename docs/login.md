Login
Used to collect a Token for a registered User.

URL : /api/v1/user/login

Method : POST

Auth required : yes

Data constraints

{
    "email": "[valid email address]",
    "password": "[password in plain text]"
}
<!-- Data example

    "email": " ",
    "password": " "
